## 실시간 채팅 데스크탑 앱

### 기술 스택

TypeScript, react, nextron.js, tailwind css, firebase(realtime DB), git

### 구현 기능

- 회원가입
- 로그인/로그아웃
- 유저 목록
- 1:1 채팅
- 그룹채팅
